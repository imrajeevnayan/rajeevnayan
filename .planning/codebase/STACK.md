# Technology Stack

## Core
- **Framework**: React 18
- **Language**: TypeScript (TSX)
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS
- **Animations**: 
  - Framer Motion (Main UI transitions)
  - GSAP 3 (Complex timeline animations, ScrollTrigger, quickSetter for cursor)
  - Three.js (3D elements via @react-three/fiber and @react-three/drei)

## Frontend Libraries
- **Icons**: Lucide React
- **Scrolling**: react-scroll
- **Cursor**: Custom GSAP-driven cursor
- **Components**: Custom glassmorphism-based UI system

## Deployment & Environments
- **Hosting**: GitHub Pages (gh-pages)
- **Base URL**: `/`

## Tooling
- **Linting**: ESLint 9+
- **CSS Processing**: PostCSS, Autoprefixer
- **Type Checking**: TypeScript 5+
