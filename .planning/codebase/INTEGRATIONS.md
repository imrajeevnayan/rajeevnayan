# Integration Endpoints

## External Services
- **GitHub API/Pages**: Deployment host and source for GitHub Calendar in the `Skills` or `Projects` section.
- **EmailJS**: Integrated for handling contact form submissions directly from the client-side.
- **Unsplash API**: Sourcing diverse images for the `Projects` section using formatting parameters like `?w=...&q=...`.

## Local Modules
- **Public Assets**: Direct serving of critical assets from the root level for performance optimization.
- **Three.js R3F Ecosystem**: Tightly integrated 3D fiber system using `@react-three/drei` helpers.
